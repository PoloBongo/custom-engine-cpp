Classe WindowModule.

Cette classe représente le gestionnaire de fen�tres dans le système. Elle hérite de la classe Module, ce qui lui permet d'être intégrée dans le syst�me de modules. Le WindowModule est responsable de la gestion et de la manipulation des fenêtres de l'application.
